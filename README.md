# LPM3 front-end #

The LPM3 is a device to measure light pollution on sea turtle nesting beaches. This repository focuses solely on the development of the front-end of the device, which can be used separately by any programmable device with an IÂ²C and a SPI interface, such as an arduino, raspberry pi, or a suitable USB adapter.
